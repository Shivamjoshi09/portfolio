Server-Side GTM

Server-Side GTM Migration for an E-Commerce Brand

Problem Statement

Client was losing significant conversion data due to third-party cookie restrictions and ad blockers affecting client-side tracking accuracy.

Solution Implemented

Architected and deployed a server-side GTM container hosted on Google Cloud. Migrated GA4 and Facebook Pixel tags from client-side to server-side. Configured first-party cookies with a custom domain to extend cookie lifetime. Implemented Google Consent Mode v2 integrated with OneTrust CMP via GTM consent triggers.

Outcome

Improved data completeness, reduced tag-related page load impact, and achieved full GDPR compliance for EU traffic.

Adobe WebSDK & AEP

Adobe Analytics and AEP Implementation for a Financial Services Client

Problem Statement

Client had fragmented data collection across web and mobile with no unified customer view and inconsistent event naming across platforms.

Solution Implemented

Designed XDM schemas in Adobe Experience Platform to standardise event taxonomy across web and iOS/Android apps. Implemented tracking via WebSDK on web and MobileSDK on native apps. Configured Edge Network datastreams, identity stitching using ECID, and Destinations for downstream activation. Built Adobe Analytics Workspace reports with calculated metrics and scheduled delivery.

Outcome

Unified cross-platform customer data, eliminated naming inconsistencies, and delivered a single source of truth for marketing and product teams.

GA4 Migration

Universal Analytics to GA4 Migration and Reporting Rebuild

Problem Statement

Client faced the UA sunset deadline with no GA4 implementation in place and stakeholders relying on UA custom reports that had no GA4 equivalent.

Solution Implemented

Conducted a full audit of existing UA tracking. Rebuilt event taxonomy in GA4 using an event-based model. Configured GA4 via GTM with custom dimensions, metrics, and enhanced eCommerce. Rebuilt all key reports as GA4 Explorations including funnel analysis, path analysis, and cohort reports. Configured GA4 audiences for Google Ads remarketing.

Outcome

Zero reporting downtime during the UA sunset transition. Stakeholders onboarded to GA4 dashboards with equivalent and improved visibility compared to UA.
